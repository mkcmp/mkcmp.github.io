What auditors ask us

How deterministic is “deterministic”?

Every replay bundles state deltas, mission JSON, and control patches. The checksum in demo/demo_pack/manifest.json matches the sealed replay_id, so third parties reproduce min-sep/NFZ stats independently.

Can I self-host the planner + shield?

Yes. Both components run on-edge (no cloud callbacks). Set ACC_BACKEND=gpd and the CLI boots gym-pybullet-drones with the same mission and control template shipped in the evidence pack.

How do you enforce the contract at runtime?

The shield tracks planner budgets (latency, comms, NFZ, min-sep). Overruns trigger drop-late or intervention, and every event lands in events.jsonl with a signature for replay.
